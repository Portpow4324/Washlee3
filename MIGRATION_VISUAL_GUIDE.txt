## Code Pattern Changes

```
┌─────────────────────────────────────────────────────────────┐
│            FIREBASE PATTERN (OLD) → SUPABASE (NEW)          │
├─────────────────────────────────────────────────────────────┤

PATTERN 1: Reading Data
─────────────────────────
OLD:
  import { getFirestore, doc, getDoc } from 'firebase/firestore'
  const docSnap = await getDoc(doc(db, 'users', userId))
  const data = docSnap.data()

NEW:
  import { supabaseClient } from '@/lib/supabaseClient'
  const { data } = await supabaseClient
    .from('users')
    .select('*')
    .eq('id', userId)
    .single()


PATTERN 2: Writing Data
─────────────────────────
OLD:
  import { setDoc, doc } from 'firebase/firestore'
  await setDoc(doc(db, 'users', userId), userData)

NEW:
  await supabaseClient
    .from('users')
    .upsert({ id: userId, ...userData })


PATTERN 3: Querying Multiple Records
──────────────────────────────────────
OLD:
  import { query, collection, where, getDocs } from 'firebase/firestore'
  const q = query(collection(db, 'orders'), where('userId', '==', id))
  const snapshot = await getDocs(q)
  const orders = snapshot.docs.map(doc => doc.data())

NEW:
  const { data: orders } = await supabaseClient
    .from('orders')
    .select('*')
    .eq('user_id', id)


PATTERN 4: Deleting Data
────────────────────────
OLD:
  import { deleteDoc, doc } from 'firebase/firestore'
  await deleteDoc(doc(db, 'orders', orderId))

NEW:
  await supabaseClient
    .from('orders')
    .delete()
    .eq('id', orderId)

└─────────────────────────────────────────────────────────────┘
```
