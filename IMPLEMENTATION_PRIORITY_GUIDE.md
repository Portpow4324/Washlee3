## 💡 DECISION MATRIX

### When choosing between features, ask:

1. **Revenue Impact**
   - Does this feature generate revenue? (0-5)
   - Does it reduce churn? (0-5)
   - Does it increase AOV? (0-5)

2. **User Impact**
   - Does this improve user experience? (0-5)
   - Does this reduce support tickets? (0-5)
   - Does this increase engagement? (0-5)

3. **Technical Impact**
   - How complex is this? (1-5, 5=complex)
   - Does it create tech debt? (0-5, 5=high debt)
   - Does it help future features? (0-5)

4. **Business Impact**
   - Does this build competitive advantage? (0-5)
   - Does this build trust? (0-5)
   - Does this reduce risk? (0-5)

**Score = ((Revenue + User + Business) - Technical Debt) * (1 - Complexity)**

Choose features with highest score first.
