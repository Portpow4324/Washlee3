HOW IT WORKS
================================================================================

BEFORE (Synchronous):
  1. User clicks "Create Account"
  2. Firebase creates auth account (200ms)
  3. SYSTEM GENERATES EMPLOYEE ID (400ms) ← USER WAITING
  4. System creates employee profile with ID (100ms)
  5. User redirected to dashboard (50ms)
  Total: 750ms wait time ⏳

AFTER (Asynchronous):
  1. User clicks "Create Account"
  2. Firebase creates auth account (200ms)
  3. System creates employee profile WITHOUT ID (100ms)
  4. User immediately redirected to dashboard (50ms)
  Total: 350ms wait time ✅
  
  Background (User doesn't wait):
  5. Cloud Function triggered automatically (100-500ms later)
  6. Generates random 6-digit ID
  7. Checks if ID already exists (retry if needed)
  8. Updates profile with ID
  9. Sets employeeIdPending = false
  Complete! ID appears in Firestore within 1 second
