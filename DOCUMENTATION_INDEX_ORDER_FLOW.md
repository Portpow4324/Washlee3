### Key Field Names (DON'T MIX UP!)
```
✅ CORRECT: uid (used everywhere)
❌ WRONG:  userId (old, don't use)
```

### Stripe Test Card
```
Card:   4242 4242 4242 4242
Expiry: 12/34
CVC:    123
```

### The Flow (Always)
```
Payment → Webhook → Firebase → API → Dashboard
```

### What Changed This Session
```
❌ OLD: Dashboard did direct Firestore query with userId field
✅ NEW: Dashboard uses secure API endpoint with Bearer token + uid field
```

### Critical Files Modified
